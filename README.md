# Sockets chat

Typing with Node, sockets chat is local server that accept users with usernames

## **Commands**

## Start server

node server.js \[port].
Example:

```
$ node serve.js 8000
```

## End server
  `Control + C`

## Start User
node client.js \[host] \[port].
Example:

```
$ node client.js 0.0.0.0 8000
```
